# hello-discord

## Atividade — Refactoring Battle

### Rodada 1 — Escreva o pior código possível (15 min)

Cada grupo cria uma função com as seguintes regras:

- Deve ter um propósito real (validar e-mail, calcular desconto, filtrar lista...)
- Deve estar funcionando — código quebrado não vale
- Deve ser o mais difícil de entender possível: nomes ruins, lógica embaralhada, sem estrutura

> 💡 A IA pode ajudar a piorar o código. Prompt sugerido:
> _"Deixe este código funcionando mas o mais ilegível possível, sem alterar o comportamento."_

---

### Rodada 2 — Descubra e refatore (15 min)

Os grupos trocam o código entre si. Sem perguntar nada para o grupo original, vocês precisam:

- Descobrir o que a função faz
- Refatorar usando IA — o prompt é livre, mas será cobrado
- Escrever 3 testes que provem que o comportamento foi preservado

---

### Rodada 3 — Apresentação e julgamento (10 min)

Cada grupo apresenta:

- O que acharam que o código fazia
- O prompt que usaram para refatorar
- O antes e o depois

O grupo original então revela o que a função realmente fazia — e a turma decide se o comportamento foi preservado.
